자료형 (Data Type)

자료형이란 변수에 저장되는 데이터의 유형이다. 데이터의 유형에는 원시(Primitive) 데이터 타입과 객체(Ovject) 데이터 타입으로 나눌 수 있다.

자료형의 종류

타입 자료형 설명 예제
원시
데이터 타입		 숫자 (number) number 데이터는 정수, 소수, 소수점, 지수를 표현할 수 있다. var a = 100;
문자 (string) string 데이터는 ''또는 ""로 표현할 수 있다. var a = "javascript";
논리 (boolean) boolean 데이터는 참(true)과 거짓(false)을 표현한다. var a = true;
var b = false;
특수값 특수값 데이터에는 null, undefined가 해당되며, 변수를 선언하고 변수의 값을 지정하지 않았을 때 자동으로 저장되는 값이다. var a= null;
var b = undefined;
var c;
객체
데이터 타입		 배열(Array),
객체(Object),
함수(Function)		 변수에는 배열, 객체, 함수 또한 저장할 수 있다. var a= [];
var b = {};
