jQuery란?

jQuery는 자바스크립트 언어를 간편하게 사용할 수 있도록 단순화시킨 오픈 소스 기반의 자바스크립트 애니메이션 라이브러리입니다. 제이쿼리를 이용하면 문서 객체 모델과 이벤트에 관한 처리를 쉽게 구현할 수 있으며, Ajax 응용 프로그램 및 플러그인도 빠르게 개발할 수 있습니다.

jQuery의 장점과 단점

* 장점 *
1. 대부분의 브라우저에서 지원
2. HTML 문서 객체 모델(DOM)을 쉽게 조작 가능
3. CSS 스타일을 간단히 적용 가능
4. 다양한 플러그인과 많은 참고 문서
5. 보다 짧고 간단한 코드
6. 오픈라이선스

* 단점 *
점점 사용하지 않는 추세…

jQuery 사용하기

**호환성에 신경써서 jQuery버전을 선택하도록 합니다.
-버전1 : 익스플로러 6, 7, 8까지 모두 지원
-버전2 : 익스플로러 6, 7, 8 지원 중단
-버전3 : 익스플로러 지원은 버전2와 같으나 더 가볍게 개선됨

* 다운로드 방식 *
1. 공식 홈페이지에서 제이쿼리 라이브러리를 다운로드 받습니다.
2. html 문서를 제이쿼리 라이브러리와 같은 경로에 저장합니다
3. 제이쿼리 라이브러리를 연결합니다.
<script src="jquery-3.3.1.min.js"></script>

* cdn 연결 방식 *
1. 공식 홈페이지에서 제이쿼리 버전을 선택 후, cdn을 복사하여 연결합니다.
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
