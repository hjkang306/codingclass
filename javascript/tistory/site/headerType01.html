사이트 만들기

사이트를 구성하는 방법에는 다양한 유형이 있습니다. 다양한 유형들을 익혀놓으면, 사이트를 만들 때 디자인적인 요소도 챙길 수 있습니다.

헤더 유형01

헤더는 사이트의 제일 위에 위치한 부분입니다. 헤더를 어떻게 디자인하느냐에 따라 사이트의 분위기가 달라지기도 합니다. 또한, 헤더를 잘 디자인하면 사용자가 사이트를 더 편리하게 이용할 수 있으니, 신경써서 작업하도록 합시다!

디자인 하기

헤더는 로고가 위치한 구역, 다른 페이지로 이동할 수 있는 네비게이션(또는 메뉴)가 위치한 구역, 추가적으로 로그인을 할 수 있는 버튼이 위치한 구역 세가지로 나눌 수 있습니다. 각 메뉴가 조화롭게 위치하도록 간격과 글자 크기를 조정합니다.

HTML 작성하기

그 다음, 만들어 둔 디자인을 보며 시멘틱 태그를 활용해 구역에 맞는 태그와 텍스트를 작성해 줍니다. 헤더 영역은 header 태그를 사용하면 좋습니다. 한번에 다 만들기 보다는 큰 요소부터 하나씩 만들면서 css를 적용해, 중간중간 확인하는 편이 좋습니다.

HTML 코드 보기 
<header id="headerType" class="header__wrap nexon">
    <div class="header__inner">
        <div class="header__logo">
            <a href="#">web <em>site</em></a>
        </div>
        <nav class="header__menu">
            <li><a href="#">헤더 영역</a></li>
            <li><a href="#">슬라이드 영역</a></li>
            <li><a href="#">배너 영역</a></li>
            <li><a href="#">컨텐츠 영역</a></li>
            <li><a href="#">푸터 영역</a></li>
        </nav>
        <div class="header__member">
            <a href="#">로그인</a>
        </div>
    </div>
</header>

CSS 작성하기

만들어 둔 디자인을 보며, 각 요소의 CSS를 설정해줍니다. 최종 완성 후 디자인과 비교하며 다른 점을 수정해줍니다. 메뉴는 중앙에 온 것처럼 보여도 메뉴가 늘어나면 위치가 틀어질 수 있으니, text-align을 이용해 중앙에 위치를 잡아줍니다. 요소를 배치하면서 제대로 css가 적용되는지 확인하고 싶다면, 해당 요소에 배경색을 지정해 확인하도록 합니다.

CSS 코드 보기

    margin: 0;
    padding: 0;
}

    text-decoration: none;
    color: #000;
}

    width: 100%;
}

    font-weight: normal;
}

            font-style: normal;
        }
