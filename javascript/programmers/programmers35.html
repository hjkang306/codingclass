프로그래머스 1단계 - 시저 암호

🍞 문제 설명

어떤 문장의 각 알파벳을 일정한 거리만큼 밀어서 다른 알파벳으로 바꾸는 암호화 방식을 시저 암호라고 합니다.
예를 들어 "AB"는 1만큼 밀면 "BC"가 되고, 3만큼 밀면 "DE"가 됩니다. "z"는 1만큼 밀면 "a"가 됩니다.
문자열 s와 거리 n을 입력받아 s를 n만큼 민 암호문을 만드는 함수, solution을 완성해 보세요.

🍞 제한 조건

공백은 아무리 밀어도 공백입니다.
s는 알파벳 소문자, 대문자, 공백으로만 이루어져 있습니다.
s의 길이는 8000이하입니다. n은 1 이상, 25이하인 자연수입니다.

🍞 입출력 예

s n result
"AB" 1 "BC"
"z" 1 "a"
"a B z" 4 "e F d"

🍞 문제 풀이

머리를 쓰지 않고 무식하게 풀었습니다. 

function solution(s, n) {
  var answer = '';
  const eng = ['a','b','c','d','e','f','g','h','i','j','k','l','m','n','o','p','q','r','s','t','u','v','w','x','y','z','a','b','c','d','e','f','g','h','i','j','k','l','m','n','o','p','q','r','s','t','u','v','w','x','y','z'];
  for(x of s.split('')){
      if (x == " "){
          answer += x;
      } else if(x.toUpperCase() == x){
          answer += eng[eng.indexOf(x.toLowerCase()) + n].toUpperCase();
      } else {
          idx = eng.indexOf(x) + n;
          answer += eng[eng.indexOf(x) + n];
      }
      console.log(eng.indexOf(x))
  }
  return answer;
}

🍞 다른 이의 풀이

저와 비슷하지만 한결 정돈된 풀이법입니다. 배열 대신 문자열을 할 걸 그랬습니다. 그리고 소문자 모음과 대문자 모음을 따로 만들면 toUpperCase()나 toLowerCase()를 쓸 필요가없었겠네요. 

function solution(s, n) {
  var upper = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
  var lower = "abcdefghijklmnopqrstuvwxyz";
  var answer= '';

  for(var i =0; i <s.length; i++){
      var text = s[i];
      if(text == ' ') {
          answer += ' '; 
          continue;
      }
      var textArr = upper.includes(text) ? upper : lower;
      var index = textArr.indexOf(text)+n;
      if(index >= textArr.length) index -= textArr.length;
      answer += textArr[index];
  }
  return answer;
}
