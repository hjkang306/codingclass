프로그래머스 1단계 - 핸드폰 번호 가리기

🍞 문제 설명

프로그래머스 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를 가립니다.
전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 *으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

🍞 제한 조건

phone_number는 길이 4 이상, 20이하인 문자열입니다.

🍞 입출력 예

phone_number return
"01033334444" "*******4444"
"027778888" "*****8888"

🍞 문제 풀이

indexOf를 이용해 Kim이 있는 자리를 찾고 제시된 문장과 함께 출력하였습니다. 

function solution(seoul) {
    var answer = '';
    let index = seoul.indexOf("Kim");
    return answer = `김서방은 ${index}에 있다`;
}

🍞 다른 이의 풀이

findIndex()라는 메서드도 배열의 인덱스를 찾는 메서드인가 봅니다. 

const solution = (arr) => `김서방은 ${arr.findIndex(s => s === 'Kim')}에 있다`;
