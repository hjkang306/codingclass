프로그래머스 1단계 - 가운데 글자 가져오기

🍞 문제 설명

단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요.
단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

🍞 제한 조건

s는 길이가 1 이상, 100이하인 스트링입니다.

🍞 입출력 예

s return
"abcde" "c"
"qwer" "we"

🍞 문제 풀이

length를 통해 문자열의 길이를 확인하고 substr로 홀수면 가운데 글자 하나, 짝수면 가운데 두글자를 return하도록 하였습니다. 

function solution(s) {
  return s.split('').length % 2 == 0 ? s.substr(s.split('').length/2 - 1, 2) : s.substr(s.split('').length/2, 1);
}

🍞 다른 이의 풀이

비슷하지만 Math를 활용한 풀이입니다. 그리고 문자열에도 바로 length를 쓸 수 있었네요… 

function solution(s) {
    return s.substr(Math.ceil(s.length / 2) - 1, s.length % 2 === 0 ? 2 : 1);
}
