HTML 기본 문서

html 기본 문서의 <html> 요소는 <head>요소와 <body>요소로 구성되어 있으며, <head> 요소 안에는 <meta>, <title>, <link>, <style>, <script> 등의 요소들이 올 수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문서의 작성자 등을 지정.
<title> HTML문서의 제목을 지정.
<link> 외부의 CSS 파일을 HTML 문서에 연결할 때 사용.
<style> HTML 문서 내에서 스타일 (CSS)를 정의.
<script> 자바스크립트를 HTML 문서 내부에서 정의, 외부에서 자바스크립트 파일을 불러올 때 사용.

lang 속성은 스크린 리더에서 언어 설정에 따라 다르게 읽어주기 때문에 중요하다.
1, 2, 3의 경우 lang="ko"로 설정되어 있다면 '일, 이, 삼'으로 읽어주지만, lang="en"으로 설정했다면 '원. 투, 쓰리'로 읽어준다.

<meta charset="UTF-8" >

UTF-8은 대부분의 문자를 처리할 수 있는 인코딩 방식이다.

<meta name = "viewport" >

뷰포트는 문서를 모바일 해상도에 맞게 HTML문서를 조절해 주는 속성이다.
